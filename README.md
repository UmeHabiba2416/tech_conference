# 🌟 Project Introduction

Welcome to the **Future Tech Conference 2026 Landing Page** 🎉

This project is a **responsive event landing page** designed using **Tailwind CSS utility classes**.  
It demonstrates how modern web interfaces are built using **clean layouts and responsive design**.

The landing page allows users to:

✔ Learn about the conference  
✔ View event speakers  
✔ Check the schedule  
✔ Register for the event
